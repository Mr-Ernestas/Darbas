# Enterprise API

#### Get ENT accounts using AE user

Takes in an enterprise account ID and returns the account details.

|Variable|Input |
|:------:|:----:|
|Role    |API_AE|
|Method  |GET   |
|Service |GET ENT ACCOUNTS|
